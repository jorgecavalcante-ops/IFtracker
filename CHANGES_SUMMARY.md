## 🔒 Configuração de Segurança

### ✅ CORS & Bloqueio de Navegador

**Status**: Nenhum problema esperado

**Por quê:**
- PDFs estão servidos **localmente** junto com o projeto
- Não há requisições cross-origin
- Tudo opera na mesma origem: `http://localhost:3000`
- pdf.js.worker está configurado via CDN (cdnjs.cloudflare.com)

**Código em equipment.js:**
```javascript
pdfjsLib.GlobalWorkerOptions.workerSrc = 
  `https://cdnjs.cloudflare.com/ajax/libs/pdf.js/${pdfjsLib.version}/pdf.worker.min.mjs`;
```
